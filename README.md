# Deduplicated OECD Cartels Database

This repository contains deduplicated versions of the OECD cartels dataset.

The filenames contain the deduplication column from the data.

Repeat entries appear to be common due to issues like multiple jurisdictions, or parent companies, which means if you care about cartels rather than the firms within them, a deduplication strategy is necessary.

- The source data can be found 
[here](https://data-explorer.oecd.org/vis?tm=Private%20International%20Cartels&pg=0&snb=3&isAvailabilityDisabled=false&page=0&df[ds]=dsDisseminateFinalDMZ&df[id]=DSD_CARTEL_QDD%40DF_QDD_CARTEL&df[ag]=OECD.DAF.COMP&df[vs]=1.0&dq=..._T....PR&to[TIME]=false).
- The guide to the data is [here](https://www.oecd.org/content/dam/oecd/en/topics/policy-sub-issues/competition-and-international-co-operation/guide-to-the-oecd-international-cartels-database.pdf). Some interpretation and cross-checking in the 
online 
data access tool is required.
